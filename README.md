# Compass Consult Website

Official website for Compass Consult -- a professional consultancy
service website built using HTML, CSS, and JavaScript.

## 🌍 Overview

This project is a fully responsive static website designed to present
Compass Consult's services, company information, and contact details in
a clean and professional format.

The site is structured for clarity, performance, and scalability, and is
suitable for deployment via GitHub Pages, shared hosting, or any static
hosting provider.

-   HTML5\
-   CSS3\

-   Responsive design (mobile-first principles)\

## 🚀 Deployment

This is a static website and can be deployed using:

-   GitHub Pages\
-   IONOS hosting\
-   Netlify\
-   Vercel\
-   Any Apache/Nginx server

### Deploying via GitHub Pages

1.  Go to repository settings\
2.  Navigate to **Pages**\
3.  Select branch (e.g. `main`)\
4.  Save

Your site will be live at:

https://your-username.github.io/Compass-Consult-Website/
